# Related Content

Dans l'application Notion, la fonctionnalité "Related Content" ou "Contenu associé" est souvent utilisée pour créer des liens entre différentes bases de données ou pages. Ceci est particulièrement utile pour organiser l'information de manière plus efficace et pour naviguer facilement entre des éléments connexes. Voici quelques façons de comment cela fonctionne :

1. **Relation entre les bases de données** : Vous pouvez créer une colonne de type "Relation" dans une base de données qui vous permettra de lier des éléments d'une base de données à une autre.
2. **Backlinks** : Notion supporte les backlinks, qui vous permettent de voir où une certaine page a été mentionnée ailleurs dans votre espace Notion.
3. **Mention de la Page** : Vous pouvez mentionner une page existante dans une autre page en utilisant le symbole "@" suivi par le nom de la page. Cela crée automatiquement un lien vers la page mentionnée.
4. **Navigation Intuitive** : Une fois que le contenu est relié, vous pouvez naviguer facilement entre différentes pages ou bases de données en cliquant sur les liens associés.
5. **Rollups** : Si vous avez établi une relation entre deux bases de données, vous pouvez utiliser la fonction "Rollup" pour afficher des données agrégées à partir de la base de données liée.
6. **Vue de Tableau Lié** : Vous pouvez créer une "Vue de Tableau Lié" qui montre un sous-ensemble des données d'une autre table, basé sur des critères spécifiques ou des relations.
7. **Mise à jour Dynamique** : Les informations mises à jour dans une base de données seront automatiquement mises à jour dans toutes les relations et rollups correspondants.
8. **Bidirectionnel** : Les relations dans Notion sont généralement bidirectionnelles. Cela signifie que si vous reliez un élément A à un élément B, B aura également un lien de retour vers A.
9. **Étiquettes des Relations** : Vous pouvez nommer les relations pour les rendre plus descriptives, ce qui est utile pour comprendre le contexte de la relation entre les différentes bases de données ou pages.
10. **Confidentialité** : Comme pour tout autre contenu dans Notion, vous pouvez contrôler qui peut voir ou éditer les relations et les rollups.

La fonction "Related Content" rend Notion extrêmement puissant pour gérer des projets complexes, organiser des bases de connaissances, et même pour des applications comme le suivi des tâches ou la gestion de la relation client.
